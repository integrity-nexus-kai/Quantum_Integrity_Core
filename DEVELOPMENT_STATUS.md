# TIG Development Status

Topological Integrity Gravity (TIG) represents the canonical physical admissibility framework of the Integrity Nexus research architecture.

The current public TIG repository contains the foundational formulation of TIG as a structurally constrained extension of General Relativity focused on horizon formation, regularized curvature behavior, and critical transition structures.

Since the initial publication state, the framework has undergone controlled structural expansion.

These later developments are maintained as mathematically separated recursive extensions.

## Current Governance Position

- TIG remains the canonical physical admissibility layer.
- SIR provides the mathematical abstraction and recursive admissibility layer.
- SGI is reserved for technical enforcement and implementation structures.
