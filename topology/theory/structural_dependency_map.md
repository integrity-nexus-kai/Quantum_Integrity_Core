# Structural Dependency Map in TIG

## Version

Draft 1.0  
Status: Structural Consolidation Phase  
Framework Status: Exploratory Mathematical Research

---

# 1. Purpose

This document defines the current structural dependency relations within the TIG framework (Topological Integrity Gravity).

The objective is not expansion of the framework.

The objective is:

- dependency clarification,
- primitive structure reduction,
- elimination of redundant concepts,
- and separation between foundational and emergent structures.

This document therefore attempts to answer:

> Which structures are fundamental, and which structures depend on them?

---

# 2. Structural Principle

The TIG framework increasingly prioritizes:

- minimal primitive structure,
- maximal internal consistency,
- and admissibility-preserving recursive evolution.

Therefore:

> Every non-primitive object should depend on a smaller set of more fundamental structures.

Unnecessary structural proliferation should be avoided.
