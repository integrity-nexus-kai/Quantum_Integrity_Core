\title{Topological Integrity Gravity (TIG):\\
A Structural Constraint on Horizon Formation in Quadratic $f(R)$ Gravity}

\author{Kai Stefan Dietrich}
\email{exionkai@proton.me}
\affiliation{Independent Researcher}

\date{\today}

\begin{abstract}
We formulate Topological Integrity Gravity (TIG) as a structural constraint on horizon formation. 
Instead of modifying the gravitational field equations directly, TIG restricts the admissible form 
of horizon conditions. In the static, spherically symmetric sector, this condition reduces to a 
cubic normal form exhibiting a saddle-node bifurcation. This introduces a critical parameter 
separating horizon-bearing and horizonless geometries. The Schwarzschild solution is recovered 
in the limit of vanishing correction scale. We analyze the resulting critical structure, derive 
universal near-critical scaling, and discuss observational implications.
\end{abstract}

\maketitle

%=====================================================
\section{Introduction}
%=====================================================

General Relativity (GR) successfully describes gravitational phenomena across a wide range 
of scales \cite{einstein1915,wald1984}. However, when extending GR or considering 
regularized geometries, the structure of horizons is no longer uniquely fixed.

Modified gravity theories such as $f(R)$ gravity introduce higher-curvature corrections 
\cite{sotiriou2010,defelice2010}. At the same time, regular black hole models demonstrate 
that singularities can be removed while maintaining horizon-like structures 
\cite{bardeen1968,hayward2006}.

These developments motivate the idea that horizon formation may be governed by a structural 
consistency condition rather than purely by dynamical equations.

%=====================================================
\section{Structural Horizon Equation}
%=====================================================

We introduce the dimensionless equation

\begin{equation}
x^3 - x^2 + \beta^3 = 0
\label{eq:tig}
\end{equation}

with

\begin{equation}
x=\frac{r_H}{2M}, \qquad \beta=\frac{r_c}{2M}.
\end{equation}

Equation \eqref{eq:tig} represents the minimal algebraic structure capable of describing 
a transition in the number of real solutions. It corresponds to the normal form of a 
saddle-node bifurcation.

%=====================================================
\section{Effective Geometry}
%=====================================================

A minimal realization is given by

\begin{equation}
F(r)=1-\frac{2Mr^2}{r^3+r_c^3}.
\end{equation}

This interpolates between Schwarzschild behavior and a regular core. The horizon condition 
$F(r_H)=0$ reduces exactly to Eq.~\eqref{eq:tig}.

%=====================================================
\section{Critical Structure}
%=====================================================

The discriminant is

\begin{equation}
\Delta=\beta^3(4-27\beta^3).
\end{equation}

Criticality occurs at

\begin{equation}
\beta_c=\left(\frac{4}{27}\right)^{1/3}, \quad x_c=\frac{2}{3}.
\end{equation}

\section{Near-Critical Scaling}
%=====================================================

Expanding around the critical point yields

\begin{equation}
\delta x \sim (\beta_c-\beta)^{1/2}.
\end{equation}

This scaling is universal for saddle-node bifurcations.

\end{equation}

\begin{equation}
\Delta_{\mathrm{ph}} \sim -\frac{16}{27}\beta^3.
\end{equation}

\section{Observational Interface}
%=====================================================

The leading-order shadow deviation is

\begin{equation}
\frac{\delta R}{R} \sim -\frac{16}{27}\beta^3.
\end{equation}

These effects are small and can be degenerate with astrophysical parameters, 
requiring combined observational constraints.

%=====================================================
\section{Conclusion}
%=====================================================

TIG provides a minimal structural constraint on horizon formation leading to 
critical behavior and universal scaling laws.

\section{Illustrative MCMC Inference Pipeline}
\label{app:mcmc}

The following minimal inference pipeline illustrates how the TIG control parameter
$\beta$ may be constrained from shadow-size measurements. It is intended as a
reproducibility example and not as a substitute for a full astrophysical analysis.

\begin{lstlisting}
import numpy as np
import emcee
import matplotlib.pyplot as plt

beta_c = (4.0 / 27.0)**(1.0 / 3.0)

def delta_shadow(beta):
    return -(16.0 / 27.0) * beta**3

def theta_tig(theta_gr, beta):
    return theta_gr * (1.0 + delta_shadow(beta))

def echo_amplitude(beta, A0=0.05):
    eps = beta_c - beta
    if eps <= 0:
        return np.inf
    return A0 / np.sqrt(eps)

data_eht = [
    {"name": "M87*",   "theta_obs": 42.0, "theta_gr": 42.0, "sigma": 3.0},
    {"name": "Sgr A*", "theta_obs": 51.8, "theta_gr": 51.8, "sigma": 2.3},
]

def log_prior(beta):
    if 0.0 <= beta < beta_c:
        return 0.0
    return -np.inf

def log_likelihood_eht(beta):
    logL = 0.0
    for d in data_eht:
        model = theta_tig(d["theta_gr"], beta)
        resid = d["theta_obs"] - model
        logL += -0.5 * (resid / d["sigma"])**2
    return logL

def log_likelihood_echo(beta, sigma_A=0.5):
    A = echo_amplitude(beta)
    if not np.isfinite(A):
        return -np.inf
    return -0.5 * (A / sigma_A)**2

def log_probability(theta):
    beta = theta[0]
    lp = log_prior(beta)
    if not np.isfinite(lp):
        return -np.inf
    return lp + log_likelihood_eht(beta) + log_likelihood_echo(beta)

ndim = 1
nwalkers = 32
nsteps = 5000
burnin = 1000

rng = np.random.default_rng(42)
initial = rng.uniform(0.001, 0.3, size=(nwalkers, ndim))

sampler = emcee.EnsembleSampler(nwalkers, ndim, log_probability)
sampler.run_mcmc(initial, nsteps, progress=True)

samples = sampler.get_chain(discard=burnin, flat=True)
beta_samples = samples[:, 0]

print("beta_c =", beta_c)
print("median beta =", np.median(beta_samples))
print("68 percent interval =", np.percentile(beta_samples, [16, 84]))
print("95 percent upper bound =", np.percentile(beta_samples, 95))

plt.hist(beta_samples, bins=50, density=True)
plt.xlabel("beta")
plt.ylabel("posterior density")
plt.tight_layout()
plt.savefig("tig_beta_posterior.png", dpi=200)
\end{lstlisting}

The numerical values used above are illustrative benchmark inputs. A full
observational analysis would require image-domain modeling, marginalization over
spin, inclination, plasma structure, accretion morphology, and gravitational-wave
systematics.
